
Метод Join() - объединяет все элементы массива в одну строку. 
    Если метод принимает строковой аргумент, то аргумент станет разделителем для элементов массива.

Метод concat() - дополняет элементы массива указанными элементами в параметрах.

Метод reverse() - меняет порядок следования элементов массива на противоположный.

Метод slice() - возвращает часть массива из указанного массива.
    //slice(a, b);
        //a - начальный индекс (включительно); 
        //b - конечный индекс (не включительно).
        
Метод splice() - универсальный метод для вставки и удаления элементов из массива. 
    Метод возвращает массив из удаленных элементов.
    
    array.splice(a, b, e1, e2, e3...);
    
    a - индекс в массиве, с которого начинать удаление.
    b - кол-во элементов, которое требуется удалить, начиная с индекса "a".
    e1, e2, e3 - добавляемые элементы в массив. Добавление начинается с позиции "a".

Метод sort() - cортирует элементы массива в алфавитном порядке.
    Для сортировки массива в каком-либо ином порядке, методу sort() можно передать свой алгоритм сортировки в виде функции сравнения:
    
    array.sort(function(a, b) { ... return number; });
    
    где a и b — два соседних элемента массива;
    number — число, указывающее методу, как должны располагаться элементы a и b:
        • если number > 0 — элементы меняются местами;
        • если number <= 0 — элементы не меняются местами;

Метод push() - добавляет элементы в конец массива и возвращает его новую длину.

Метод pop() - удаляет последний элемент из массива и возвращает удаленное значение.

Метод unshift() - добавляет элементы в начало массива и возвращает его новую длину.

Метод shift() удаляет первый элемент из массива и возвращает удаленное значение.
